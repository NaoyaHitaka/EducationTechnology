関数f(x)の定積分は、y=f(x)のグラフで囲まれた図形の面積に関係がある。

1次関数については、グラフが直線になるから、面積が調べやすい。

以下では関数f(x)=x+1について調べてみよう。

[1] 図[A]において、関数y = x + 1 のグラフとx軸およびy軸、直線x = tで囲まれた斜線部分の面積はtの関数である。

これをS(t)とすると・・・

S(t) = ½ * {1 + (t + 1)} * t

S(t) = ½ t2 + t

S'(t) = t + 1;

すなわち、S'(x) = x + 1であるから、S'(x) = f(x)が成り立つ。

[2] 図[B]において、関数y = x + 1のグラフとx軸および2直線x = a, x = bで囲まれた斜線部分の面積Sは、[1]の関数S(t)を用いると

S = S(b) - S(a)　となる。

ここで、S'(x) = f(x)であるから、Sは次のように表される。
